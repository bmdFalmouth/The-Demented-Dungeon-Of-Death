# The Demented Dungeon Of Death
’You find a game box from 80s, intriqued you open it up and find the manual, map and hand written notes. You decided to boot into an emulator and try the game out…’

‘The Demented Dungeon of Death’ is a love letter to 80s top down adventure games, when there was no digital map, you had to make use of paper maps and notes. Using these notes, you will recreate the journey of a player in the 80s and learn something about them as you go.
